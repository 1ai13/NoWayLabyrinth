## Goals

- Search for keys in hidden chests  
- Defeat mobs  
- Complete all levels to escape the maze  

---

## Controls

| Action | Keys / Input |
|--------|--------------|
| Move character | `W` `A` `S` `D` |
| Normal attack | Left Mouse Click |
| Special attack | Right Mouse Click |
| Swap element | `Space` |
