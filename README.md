# NoWayLabyrinth

### Your mage needs to survive the maze!

Demo finished, project ***not*** maintained, you can complete the game run, but some features are missing.

You can download the [game here](https://1ai-13.itch.io/no-way-labyrinth) or from the [releases](https://github.com/1ai13/NoWayLabyrinth/releases) of this repo

Game path -> build/lib/NowWayLabyrinth.jar

Only .jar available, you will need Java installed to run it the project

Goals:

- Search the keys in the hidden chests
- Kill the mobs
- Complete the levels until you escape

Controls: 

- Moving character: A-W-S-D

- Attacks: Left (normal) & right (special)

- Swap Element: Space
